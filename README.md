# Emalls Application 🛒🛍

## Installation 👨‍💻

1️⃣- Install ``emalls-application`` with npm

```bash
    git clone https://github.com/peyman2dev/emalls.git
    npm run dev
```

2️⃣- Then run the server 
```bash
    npx json-server --watch db.json --port 3000
```


    
## Tech Stack

**Client:** React JS, Redux JS, TailwindCSS
